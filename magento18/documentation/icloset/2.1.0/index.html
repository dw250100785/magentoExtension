A) What is this? - top

It is a Magento eCommerce + jQuery extension to use your clothing products on virtual models.


B) Features - top

  • Full HTML5/CSS3 power.
  • Pull selected categories/products to the widget
  • Drag and drop
  • System works with ajax requests
  • You can personalize product view.
  • You can personalize next/back controllers view.
  • You can personalize next/back controllers view.
  • You can personalize category view.
  • You can personalize total label information.

C) How does it work? - top

  1. You will need to take some pictures of your products, and also a model where clothes fit on it.
  2. Add a new Magento attribute called "image_play"
  3. Add these additional pictures to your clothing products and mark them using the new attribute "image_play"
  4. Create a new tree of categories, different that the main you use.
    This is because usually you do not want to have all products on the iCloset.
    Admin > Catalog > Manage Categories
  5. Add the products you want on the iCloset to these categories.
  6. Finally, config your iCloset.
    Admin > System > Configuration > BigHippo iCloset (left menu) > General Options
    Select main category for products.
    Select model image.

E) Customisation - top

To init your iCloset jQuery module, just use these code:

		   (function($) {
		        $(document).ready(function($) {
		            //iCloset config
		            iCloset.icWrapper = "#ic-wrapper";
		            iCloset.imgLoading = 'getSkinUrl("icloset/images/loading.gif");?>';
		            iCloset.pageSize = 12;
		            iCloset.currency = "getLocale()->currency(Mage::app()->getStore()->getCurrentCurrencyCode())->getSymbol(); ?>";
		            //Customize events
		            //...
		            //iCloset launchs
		            iCloset.init("women");
		        });
		    })(jQuery);

There are a few methods from iCloset javascript jQuery that you could easily override.

Most of them are the way as you could render the parts of the widget.

			/**
			* How to render every category on the left side.
			*/
			renderCategory : function(category, method) {
				var html = "
  • ";
				html += "";
				html += category.name + "";
				html += "
    • "; 
				return html;
			},
			
			/**
			* How to render every product on the grid.
			*/ 
			renderProduct : function(product) {
				return product.name + "
    " + iCloset.currency + iCloset.formatNumber(product.price); 
			},
			
			/**
			* How to render the total label.
			*/
			renderTotalLabel : function(totalProducts, totalPrice) {
				$("#ic-total").html("");
				if(totalProducts > 0) {
					$("#ic-total").html(totalProducts + " products " + iCloset.currency + totalPrice);
				}
			},
			
			/**
			* How to render the category label when it is selected.
			*/
			renderCategoryLabel : function(categoryId) {
				//Category name label
				$("#ic-filter-label").html(iCloset.categories[categoryId]);
				
				//Category buttons
				$("#ic-cat-").removeClass("selected");
				$.each(iCloset.categories, function(index, value) {
					$("#ic-cat-" + index).removeClass("selected"); 
				});
				$("#ic-cat-" + categoryId).addClass("selected");
			},
			
			/**
			* How to render the backward / forward controllers. And also current / total pages.
			*/ 
			renderControllers : function(backwardMethod, forwardMethod, currentPage, totalPages) {
				var htmlControllers = "";
				if(backwardMethod != null) {
					htmlControllers = htmlControllers + "< < back";
				}
				if(forwardMethod != null) {
					htmlControllers = htmlControllers + "next > >";
				}
				$("#ic-controllers").html(htmlControllers);
				var htmlPages = "";
				if(totalPages > 1) {
					htmlPages = htmlPages + "" + iCloset.page + " / " + totalPages + "";
				}
				$("#ic-pages").html(htmlPages);
			}
